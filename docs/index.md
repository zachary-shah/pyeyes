# pyeyes

**pyeyes** is an n-dimensional data visualization tool for comparing images, especially designed for MRI. It is built on [Holoviews](https://holoviews.org/), [Bokeh](https://bokeh.org/), and [Panel](https://panel.holoviz.org/) for interactive plotting.
